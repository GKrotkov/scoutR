# Teams (Statbotics)

Returns a batch of team (or team-year, or team-event, or team-match)
objects conforming to the provided parameters.

## Usage

``` r
teams_sb(..., yr = NULL, event = NULL, match = NULL)
```

## Arguments

- ...:

  Optional parameters passed to v3/teams endpoint

- yr:

  (optional) year of interest

- event:

  (optional) event of interest

- match:

  (optional) match of interest
