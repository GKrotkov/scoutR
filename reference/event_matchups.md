# Event Matchups

Returns a list containing all a team's partners and opponents for a
particular event.

## Usage

``` r
event_matchups(event_code, team_id)
```

## Arguments

- event_code:

  TBA-legal event code (ex. "2025chcmp")

- team_id:

  team of interest as int or TBA id (ex. "frc449", 449)
