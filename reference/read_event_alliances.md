# Read event alliances from key

Returns an event alliances object given an event key

## Usage

``` r
read_event_alliances(key)
```

## Arguments

- key:

  TBA legal event key

## Value

TBA Alliances object

## Author

Gabriel Krotkov

## Examples

``` r
read_event_alliances("2015paphi")
#> Error in read_event_alliances("2015paphi"): could not find function "read_event_alliances"
```
