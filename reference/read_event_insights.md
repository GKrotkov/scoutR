# Read Event Insights from key

Returns event insights given event key

## Usage

``` r
read_event_insights(key)
```

## Arguments

- key:

  TBA legal event key

## Value

TBA Insights object

``` r
read_event_insights("2018pawch")
#> Error in read_event_insights("2018pawch"): could not find function "read_event_insights"
```
