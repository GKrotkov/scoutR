# Event District Points

Reads event district points

## Usage

``` r
event_district_points(key, tiebreakers = FALSE)
```

## Arguments

- key:

  TBA legal event key

- tiebreakers:

  (bool) unpack tiebreakers?

## Value

tidy tibble of district points?
