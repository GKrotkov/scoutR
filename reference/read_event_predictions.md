# Read Event Predictions

Returns event predictions given event key

## Usage

``` r
read_event_predictions(key)
```

## Arguments

- key:

  TBA legal event key

## Value

TBA Predictions object

## Author

Gabriel Krotkov

## Examples

``` r
read_event_predictions("2018mrcmp")
#> Error in read_event_predictions("2018mrcmp"): could not find function "read_event_predictions"
```
