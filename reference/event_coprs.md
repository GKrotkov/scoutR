# Get Event cOPRs

Wrapper function for read_event_coprs and tidy_event_coprs

## Usage

``` r
event_coprs(key, digits = 2)
```

## Arguments

- key:

  (string) event key

- digits:

  number of digits to round the result to

## Details

TBA exports cOPRs for events as early as 2023. If you want cOPRs for
earlier events, you can compute them with \`fit_event_lr\` and a wise
response choice
